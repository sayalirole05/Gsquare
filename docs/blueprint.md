# **App Name**: Gsquare

## Core Features:

- Static Content Display: Hardcoded content for all corporate pages, including Home, About, Services Hub, detailed service pages, Industries, Process, Pricing, Case Studies, and Contact, ensuring zero external dependencies.
- Client-Side Routing: Seamless client-side navigation between all pages, mimicking a multi-page application experience within a static build without server-side routing or configuration.
- SEO Optimization Toolkit: Integration of static SEO elements including unique page titles, meta descriptions, H1 tags, Open Graph tags, and JSON-LD structured data on each page to maximize search engine visibility.
- Responsive Layout: Mobile-first design approach with a stack layout on mobile devices, ensuring optimal viewing across all screen sizes, including a hamburger navigation for mobile.
- Static Contact Form: A contact form that triggers a client-side mailto action, displaying business Name, Address, and Phone (NAP) without requiring any backend or API calls.
- Enterprise-Grade Animations: Subtle, performant CSS-only animations such as fade-in on scroll, card hover lift effects, button hover transitions, and a navbar shadow on scroll, respecting 'prefers-reduced-motion' settings.

## Style Guidelines:

- Primary color: `#2E86DE` (a rich, professional blue, HSL 210, 71%, 52%) chosen for its representation of corporate reliability and clarity.
- Background color: `#EBF3FA` (a soft, desaturated blue, HSL 210, 15%, 95%) providing a clean and understated canvas that complements the primary color.
- Accent color: `#00BFA6` (a vibrant teal, HSL 172, 100%, 37%) used for calls to action and highlights, offering a refreshing contrast to the primary palette, echoing user preference for secondary branding.
- All text uses 'Inter', a modern sans-serif typeface, providing excellent readability for both corporate headings (bold, clear hierarchy) and body text (16px with 1.6 line-height, color #2F3E46). Note: currently only Google Fonts are supported.
- Inline SVG outline icons, drawn from libraries like Feather, Heroicons, or Lucide. All icons will be 24px with a stroke color of `#2E86DE`, ensuring a consistent, clean, and corporate visual style without any filled elements.
- A 'clean white card' design philosophy with subtle shadows, structured with corporate spacing guidelines for a polished and spacious user interface. Mobile-first stacking ensures optimal adaptability, with a hamburger menu for navigation on smaller screens.
